# ✈️ SkyWings Analytics Pro Dashboard

## 📊 Overview
SkyWings Analytics Pro Dashboard is a web-based analytics application that helps visualize travel booking data and generate business insights.

## 🚀 Features
- Upload CSV data
- Total Revenue, Bookings & Average Value
- Monthly Revenue Chart
- Peak vs Off-Peak Analysis
- Top Destinations Visualization
- Responsive UI

## 🛠️ Tech Stack
- HTML
- CSS
- JavaScript
- Chart.js

## 📁 How to Run
1. Download or clone the repository
2. Open index.html in your browser
3. Upload your CSV file

## 📊 Sample Data Format
Date,Revenue,Destination,Season
